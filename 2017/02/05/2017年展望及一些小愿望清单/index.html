2017年展望及一些小愿望清单

转眼又开始了忙碌的工作，虽然工资涨了点，但是周末也变成了单休，地点也从市里变成了远郊。

新项目有很多都还没仔细研究的框架，有的甚至只知道名字，任重而道远。希望自己能尽快熟悉起来，也算是项目驱动人生吧。

2016年给自己列了好多愿望清单，但是基本都没实现。从下半年开始就各种不太顺，导致经济上和精力上都没有能力去完成。

2017年，特地去了一次雍和宫拜佛烧香，心诚则灵吧。现在看来可以把去年的愿望清单直接改下时间就好了\手动捂脸。
