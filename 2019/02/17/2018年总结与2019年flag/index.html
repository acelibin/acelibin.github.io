2018年总结与2019年flag

2018过的很忙很忙，忙得没有时间思考，像一个无头苍蝇，天天转啊转，好累。很多目标都没有完成，唯一值得欣慰的是终于考下来驾驶证，算是最大的收获吧。2019还是继续去年没做完的事情，争取全部完成。flag还是要立的，万一实现了呢。

2018年总结

  • 最大的收获：拿到驾驶证，但是离真正会开车还是有很大距离，过马路恐惧症估计不会好了。学习开车是一个无比痛苦的经历，想起来小时候不会加减乘除被妈妈骂的狗血淋头，希望不会再有第二次了。

  • 最自豪的事情：自己一个人第一次坐飞机就是去马尼拉-长滩岛，在人生地不熟的情况下玩了8天。出国游也不是那么遥远了，作为海岛控，争取以后每年都出去玩一次。

  • 忙就一个字：真是不知道为什么，每天有那么多忙不完的事情。单纯的忙还好，忙得焦头烂额就不好了。很多东西明明可以做的更好，但是因为时间紧迫只能就那么结束，心有不甘。

别的真是没有什么了，连总结都这么空洞。我其实还算是一个靠谱的人啊，现在越来越觉得自己不靠谱了。做的东西bug很多，细节方面还有很多优化的地方。

2019年目标

学海无涯乐为舟：

  • 学会 ES6、TypeScript

  • 加强 JS、CSS 基础

  • 学会 React 并了解原理

  • 试着用 Express 和 Mongoose 开发一个小程序

  • 《你不知道的JavaScript》

  • 《CSS设计指南》

  • 港澳游

挑战自己

  • 徒步沙漠

  • 爬华山

get√ 新技能

  • 练习硬笔书法

  • 学会用SuperPads弹奏5首曲子

其它

英语真的是一个非常重要的工具，这次出国也深深的发现自己的不足，大部分听得懂，但是想要开口说还是挺难，不知道怎么表达才是正确的。想想在英语不怎么听得懂的情况下，一个人完成了海底漫步，还是很有挑战性的。所以还是要多学习英语，多听多看多说，不然以后出去还得被英语所困。

新的一年，希望自己充实一些，多多学习，多多思考，享受工作，享受生活。
